# Bioinformatics Coursework 2026

## Overview

This repository contains coursework, reproducible workflows, reports, and analyses developed throughout a graduate-level bioinformatics course focused on computational genomics, transcriptomics, statistical analysis, comparative genomics, and biological data interpretation.

The repository is organized into modular sections corresponding to the course modules, with emphasis on:

* Reproducible computational workflows
* RNA-seq analysis pipelines
* Statistical analysis of omics data
* Biological interpretation of genomic datasets
* Data visualization and reporting
* Comparative and functional genomics

---
